# Comparable Company Analysis (Comps)

A relative-value method: value a company by reference to the trading multiples of a peer group of public comparables. The standard counterpart to [[dcf]] (which is absolute / intrinsic).

## Standard multiples

- **EV / Revenue** — earliest-stage / pre-profit comparison
- **EV / EBITDA** — most common for mature operating companies (capital-structure-neutral)
- **EV / EBIT** — controls for D&A differences
- **P/E** — equity-level, post-tax
- **P / Book** — financials and asset-heavy businesses

## Workflow

1. Define peer set (sector + size + business model + growth profile)
2. Pull operating metrics (revenue, growth, margins, leverage) and multiples
3. Compute peer-set medians / quartiles
4. Apply selected multiple range to the target's metric → implied valuation range
5. Flag outliers and explain why (premium / discount drivers)
