# Returns Analysis

**Plugin**: [[plugin-private-equity]]
**Triggers**: "returns analysis", "IRR sensitivity", "MOIC table", "what's the return at", "model the returns", "back of the envelope"

Build quick IRR/MOIC sensitivity tables for PE deal evaluation. Models returns across entry multiple, leverage, exit multiple, growth, and hold period scenarios. Used to size up a deal, stress-test assumptions, prepare IC returns exhibits.
