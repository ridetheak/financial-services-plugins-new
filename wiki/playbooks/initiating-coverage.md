# Initiating Coverage

**Plugin**: [[plugin-equity-research]]
**Triggers**: institutional first-time coverage requests

Create institutional-quality equity research **initiation reports** through a structured 5-task workflow with strict prerequisite ordering:

1. **Company research** — qualitative + business model deep-dive
2. **Financial modeling** — build the [[3-statement-model]]
3. **Valuation analysis** — [[dcf]] + [[comps-analysis-concept]] + precedent transactions
4. **Chart generation** — exhibits for the report
5. **Final report assembly** — DOCX output following JPM/GS/MS format

Tasks 3-5 have hard dependencies on earlier tasks.
