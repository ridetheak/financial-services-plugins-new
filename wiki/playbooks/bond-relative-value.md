# Bond Relative Value

**Plugin**: [[plugin-lseg]] (LSEG)
**Triggers**: bond richness/cheapness analysis, spread decomposition, bond comparison, bond value vs curves, rate shock scenarios

Perform relative value analysis on bonds by combining pricing, yield curve context, credit spreads, and scenario stress testing. Decompose total spread into risk-free + credit + residual components — the residual reveals true richness/cheapness. Stress test with scenarios to confirm view.
