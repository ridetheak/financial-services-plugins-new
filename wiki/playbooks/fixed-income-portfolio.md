# Fixed Income Portfolio

**Plugin**: [[plugin-lseg]] (LSEG)
**Triggers**: reviewing bond portfolios, computing portfolio duration and DV01, cashflow waterfalls, rate scenarios, portfolio composition

Review fixed income portfolios by pricing multiple bonds, retrieving reference data, analyzing cashflows, running scenario analysis. Aggregates bond-level analytics into portfolio-level metrics (market-value-weighted yield, duration, convexity). Frames relative to benchmark when available.
