# 部门概览

描述：创建全面的行业和部门景观报告，涵盖市场动态、竞争定位、关键参与者和主题趋势。用于客户请求、部门启动、主题研究或内部知识构建。触发条件："部门概览"、"行业报告"、"市场景观"、"部门分析"、"行业深度分析"或"主题研究"。

## 工作流程

### 第 1 步：定义范围

- **部门/子部门**：什么行业以及如何明确定义？
- **目的**：客户报告、内部研究、推介材料、创意生成
- **深度**：高级概览（5-10 页）或深度分析（20-30 页）
- **角度**：中立景观vs主题论文（例如，"AI 基础设施建设"）
- **范围**：仅公开公司还是包括私人公司？

### 第 2 步：市场概览

**市场规模和增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5 年复合增长率）
- 预测增长率和关键假设
- 市场细分（按产品、地理、最终市场、客户类型）

**行业结构**
- 分散vs整合 — 前 5 大市场份额
- 价值链图 — 价值在哪里累积？
- 业务模型类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势和驱动因素**
- 长期增长趋势（3-5 个主要趋势）
- 逆风和风险
- 技术中断向量
- 监管发展
- M&A 活动和整合趋势

### 第 3 步：竞争景观

**公司概况**（前 5-10 大参与者）：

| 公司 | 收入 | 增长 | EBITDA 利润率 | 市场份额 | 关键差异 |
|------|------|------|-------------|---------|---------|
| | | | | | |

对于每家公司，简要概况：
- 业务描述（2-3 句）
- 战略定位和护城河
- 最近发展（盈利、M&A、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在获得/失去市场份额及其原因？
- 来自新进入者或相邻参与者的破坏风险

### 第 4 步：估值背景

- 部门交易倍数（当前和历史范围）
- 溢价/折扣驱动因素（增长、利润率、市场定位）
- 最近 M&A 交易倍数
- 部门与更广泛市场的比较？

### 第 5 步：投资影响

- 最佳风险/收益机会在哪里？
- 什么主题赌注可以通过这个部门表达？
- 部门内的关键辩论（牛市vs熊市论证）
- 可能改变部门叙述的催化剂

### 第 6 步：输出

- Word 文档或 PowerPoint，包含：
  - 市场概览和规模
  - 竞争景观图
  - 公司比较表
  - 估值总结
  - 关键图表：市场增长、份额趋势、估值历史
- 详细公司数据的 Excel 附录

## 重要说明

- 所有市场规模数据的来源 — 引用研究公司或方法
- 区分 TAM 炒作和现实可寻址市场
- 部门概览老化很快 — 注明日期并标记可能过时的数据
- 图表很必要 — 市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果用于客户，将"那又怎样"调整到他们的具体情况（M&A 目标识别、竞争定位、市场进入）
